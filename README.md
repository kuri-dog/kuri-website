# Kuri Site

New static website build for `kuri.co.nz`.

## Stack

- Astro
- Netlify
- GitHub

## Workflow

1. Make changes in a branch.
2. Netlify creates a Deploy Preview.
3. Review the preview.
4. Merge approved changes to production.
